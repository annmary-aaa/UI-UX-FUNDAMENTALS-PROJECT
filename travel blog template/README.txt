Wanderlust Tales - Multi-page Travel Blog Template (Demo)
--------------------------------------------------------
What is included:
- index.html (home page)
- posts/* (6 example post pages)
- destinations/* (4 category pages)
- about.html, contact.html
- css/style.css (shared styles)
- js/main.js (interactive behavior: back-to-top, modal previews)
- images/posts/* (placeholder images - replace with real images)

How to use:
1. Unzip and open index.html in your browser.
2. Replace placeholder images in images/posts/ with real jpg/png files.
3. Edit posts/*.html to update content.
